# 📦 Personal_Box

> 🧰 一个用于整理和保存个人项目、脚本、配置文件与笔记的多功能仓库。

## 🪄 简介

\`Personal_Box\` 是我的个人仓库，用于集中管理：

- 💻 常用代码片段（C / Python / LaTeX / Bash 等）
- ⚙️ VS Code / Git 等配置文件
- 📄 学习笔记、课程实验与文档
- 🧠 小工具脚本与项目原型
- 📚 个人经历、相关文件

目标是构建一个「个人知识与工具中枢」，方便同步、复用和长期维护。

## 🗂️ 仓库结构（规划）

```text
Personal_Box/
├── scripts/          # 实用脚本与命令工具
├── notes/            # 学习笔记与文档
├── config/           # 配置文件（VS Code、Git、LaTeX 等）
├── projects/         # 小型实验 / Demo 工程
└── README.md
```


> 实际结构会随着内容增加逐步调整。

## 🚀 使用说明

克隆仓库：
推荐使用SSH，部分文件是大文件。

按需查看各目录内容即可。

## 🧩 可能用到的环境

- Git 2.x+
- Python 3.x（若使用相关脚本）
- VS Code / Keil / Proteus / STM32 等开发环境（视具体项目而定）
